WHO AM I?

My name is Sayuj Stephen. I am currently a third year Mechanical Engineering Student at the University of Calgary. Over the past three years I've been curious about different things and tried my hands at projects to further develop my understanding and have some fun. Below is the culmination of those projects

ESP 32 GARDEN

After making a Garden using the Arduino Uno, this was my first project using the ESP 32 microcontroller. I was able to use this microcontroller to control a water pump, along with lights and monitor temperature and humidity coniditons of the soil.

Future Plans

I plan to utilize the ESP 32's bluetooth compatibility to design an app where the lights and watering features can be controlled using the app and all soil information can be displayed directly into the app.

Arduino Mega Game Console

In this project I utilized the Arduino mega and Arduino mega libraries to design game that can be played on this console. Designed using SolidWorks.

Blinds Brackets

Part designed for Stephens Blinds & Shades this is a bracket attachment for Korean Zebra Blinds. Through careful measurement and multiple design iterations these serve as a perfect replacemetn for existing brackets. However their structural intergrity due to 3D printing is volatile and I am currently in the process of trying out new materials in an effort to improve strength and durability of the parts.

ESP 32 ROVER

Currently my biggest project yet. This project aims to utilize the ESP32 to its full extent, controlling 4 independent motors using motor controllers. This aim of this project is to create a rover that can be remotely controlled but also traverse its environment autonomously. Currently the rover is able to move thanks to a connection with a joystick.

RC CAR

This is the latest iteration of the RC car. Unfortunately due to a bug I lost most of the files to the orignal version of the RC car. Currently this model aims to integrate metal gears into the system to improve the logevity of the system as the previous model had used plastic gears which did not last long due to the high RPM of the motors.
